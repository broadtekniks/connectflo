# Integration Guides

ConnectFlo supports various integrations to enhance your customer service workflows. This directory contains setup guides for each integration.

## Available Integrations

### CRM & ERP

- [HubSpot](./hubspot.md) - Sync contacts, companies, and deals with HubSpot CRM
- [Salesforce](./salesforce.md) - Connect to Salesforce for customer relationship management
- [Zoho CRM](./zoho.md) - Integrate with Zoho CRM for contact and deal management
- [Odoo](./odoo.md) - Connect to Odoo ERP system

### Google Workspace

- [Google Calendar](./google-calendar.md) - Schedule meetings and manage appointments
- [Gmail](./gmail.md) - Access email communications
- [Google Drive](./google-drive.md) - Store and retrieve files
- [Google Sheets](./google-sheets.md) - Access spreadsheet data

## Integration Features

All integrations support:

- **Two-way sync** - Changes sync bidirectionally between ConnectFlo and the connected system
- **Real-time updates** - Data is synced in real-time or near real-time
- **Secure authentication** - All credentials are encrypted using AES-256-GCM
- **Dynamic field discovery** - Available fields from your CRM are automatically discovered
- **Workflow integration** - Use CRM data in workflow conditions and actions
